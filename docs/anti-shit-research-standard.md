# 反屎文研究标准

## 一、什么叫有效样本

这里的“有效样本”不是“好文章”，而是适合进入研究库、能被比较的内容。

满足以下条件才进入研究库：

1. 家居 / 生活相关
2. 至少 1 张图
3. 标题长度 8-40 字
4. 正文字数 120-1500 字
5. 不是纯广告导流 / 群引流 / 直播口播壳
6. 能提取到标题、正文、图片数、链接这些基础字段

## 二、什么叫 💩 文

V1 不用“审美差”定义 💩 文，而用下面这些更可解释的信号：

1. 情绪词很多，信息很少
2. 标题承诺很大，正文兑现很少
3. 很像在分享经验，实际上没有具体步骤 / 预算 / 尺寸 / 前后对比
4. 图在撑场面，文在拖时间
5. 模板味重，像拼热词
6. 看起来很会写，读完却拿不走任何东西

## 三、V1 的 💩 味指标

### 1. 空话味
过度使用：
- 氛围感
- 高级感
- 松弛感
- 幸福感
- 治愈

### 2. 模板味
常见句式：
- 普通人直接抄作业
- 一定要学会
- 闭眼入
- 谁懂

### 3. 标题诈骗味
标题非常会喊，但正文缺乏：
- 结果
- 过程
- 条件
- 数据

### 4. 去味儿能力
好文章不一定高级，但至少能留下：
- 一个具体场景
- 一个真实限制
- 一个可验证细节

如果三者都没有，就更接近 💩 文。
